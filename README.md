# 专升本心理学刷题 App

这是一个可直接打开使用的前端刷题工具，支持：

- 导入 Word 题库（`.docx`）或文本题库（`.txt`）
- 自动识别单选、多选、判断、简答/论述题
- 自动识别章节并按章节筛选
- 顺序/随机练习
- 模拟考试（可设置题量与时长、倒计时、答题卡跳转、可回看改答案、交卷页自动出分）
- 错题本可导出 TXT / Word
- 错题本与本地进度统计（浏览器 `localStorage`）

## 1. 如何使用

1. 直接双击打开 `index.html`（或用本地静态服务打开）。
2. 在“导入题库”处选择你的 `.docx` 或 `.txt` 文件。
3. 点击“导入并生成题库”。
4. 选择练习模式后点击“开始练习”。
5. 项目内置了已整理的 `seed_bank.json`（当前 219 题，首次打开会自动导入一次）。
6. 若需要模拟考试，切换“练习类型”为“模拟考试”，设置题量与时长后开始。
7. 考试过程中可通过“答题卡”点击题号快速跳转，并按“全部/已答/未答”筛选查看。
8. 考试过程中可用“上一题/下一题”回看并修改答案，交卷前可反复保存。
9. 若要撤销某题作答，可点击“清空本题答案”恢复为未作答。
10. 交卷后可在成绩页按“全部/仅错题/仅未答/仅正确”筛选明细。
11. 考试过程中可手动“交卷”，也可等待倒计时结束自动交卷。
12. 在“错题本”区域可一键导出 `TXT` 或 `Word(.doc)`。

## 2. 推荐题库格式（Word）

建议按下面结构整理（题型标题可选，但推荐写）：

```text
单选题
1. 心理学研究对象是？
A. 心理现象及其规律
B. 生理结构
C. 社会制度
D. 经济活动
答案：A
解析：心理学主要研究心理现象及其活动规律。

2. 感觉适应是指？
A. 感受性提高
B. 感受性降低或变化
C. 注意力转移
D. 记忆增强
答案：B

多选题
3. 下列属于注意品质的是？
A. 注意广度
B. 注意稳定性
C. 注意分配
D. 注意转移
答案：ABCD
解析：注意品质包括广度、稳定性、分配和转移。

判断题
4. 心理是脑的机能。
答案：对
解析：唯物主义心理观认为心理是脑的机能。
```

如果你希望按章节筛选，建议在 Word 中增加章节标题，例如：

```text
第一章 普通心理学
第二章 发展心理学
章节：教育心理学
【第三章 社会心理学】
```

## 3. 导入规则说明

- 题号支持如：`1.` `1、` `1)` `第1题`
- 选项支持 `A.` `A、` `A)` 等格式
- 答案行支持：`答案：A` / `答案：AC` / `答案：对`
- 解析行支持：`解析：...`
- 判断题答案可写：`对/错`、`正确/错误`、`√/×`、`T/F`
- 章节标题支持：`第X章...`、`章节：...`、`【第X章...】`

## 4. 注意事项

- `.docx` 解析依赖在线加载 `mammoth` 脚本，首次使用需要联网。
- 导入新题库会覆盖旧题库并重置进度/错题本。
- 数据只保存在当前浏览器本地，不会上传。
