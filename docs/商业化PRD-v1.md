# 专升本心理学刷题 App 商业版 PRD v1

## 1. 产品定位
- 产品名称：专升本心理学刷题
- 核心场景：专升本心理学备考用户的每日刷题、考前冲刺、错题复盘
- 一句话价值：用最短时间把“会做题”变成“能拿分”

## 2. 商业目标（首个 90 天）
- 目标 1：完成可付费闭环（注册 -> 试用 -> 付费 -> 续费）
- 目标 2：验证付费模型可行性（付费率 >= 3%）
- 目标 3：建立题库生产和质量流程（每周可稳定更新）

## 3. 用户与分层
- 核心用户：专升本考生（心理学科目）
- 用户分层：
  - 轻度：只做免费题，低频访问
  - 中度：每日练习，关注提分
  - 重度：冲刺备考，愿为效率和服务付费

## 4. 收费与权益设计
- 免费版：
  - 每日可做题 30 道
  - 支持基础错题本
  - 不支持 AI 简答评分
- 会员版（建议）
  - 月卡：29 元
  - 季卡：79 元
  - 年卡：199 元
  - 权益：不限题量、智能组卷、冲刺包、AI 解析与简答建议、云端同步
- 增值服务
  - 7 天冲刺营、人工批改、1v1 学习计划

## 5. MVP 功能范围（必须上线）
- 账号与身份
  - 手机号/微信登录
  - 设备间数据同步
- 会员与支付
  - 会员购买、到期、恢复购买
  - 权益开关（免费/会员）
- 学习核心
  - 章节练习、随机练习、模拟考试
  - 错题本、收藏、复习计划
- 简答题能力
  - 用户作答保存
  - 参考答案展示
  - AI 评分建议（MVP 可先做“弱评分 + 关键点缺失提示”）
- 题库运营后台（最简）
  - 导入、审核、发布、下线

## 6. 非功能需求
- 可用性：核心流程成功率 >= 99%
- 性能：首页 2 秒内可交互，答题切题 < 300ms
- 安全：登录态、接口鉴权、支付签名校验
- 合规：隐私协议、用户协议、未成年人提示、APP 备案流程准备

## 7. 页面与关键流程
- 首次启动
  - 展示产品价值 -> 免费试用权益 -> 引导注册
- 刷题流程
  - 选章节/模式 -> 作答 -> 反馈 -> 错题沉淀
- 支付转化流程
  - 命中免费额度上限 -> 展示会员权益页 -> 支付 -> 权益即时生效
- 留存流程
  - 每日计划提醒 -> 连续打卡 -> 周报

## 8. 数据指标与埋点
- 漏斗指标
  - UV -> 注册转化率 -> 首次开练率 -> D1/D7 留存 -> 付费率
- 学习指标
  - 日人均题量
  - 模拟考完成率
  - 错题二次正确率
- 商业指标
  - ARPPU
  - CAC
  - LTV
  - 退款率

## 9. 验收标准（首版）
- 用户可完成注册、购买、恢复购买
- 免费与会员权益严格隔离
- 题库可从后台发布并在前端可见
- 核心埋点数据可在看板看到日级趋势

## 10. 暂不做（避免分散）
- 全科目扩张
- 复杂社交功能
- 大而全的内容社区

## 11. 版本节奏
- v1（0-4 周）：付费闭环 + 账号 + 基础会员权益
- v1.1（5-8 周）：AI 简答建议 + 题库后台优化
- v1.2（9-12 周）：增长实验 + 价格/权益 AB 测试
