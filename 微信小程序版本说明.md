# 微信小程序版本说明

已新增微信小程序版本目录：

- `/Users/apple/Downloads/专升本/wechat-miniprogram`

主要能力：

- 内置题库（读取 `wechat-miniprogram/data/seed_bank.js`）
- 章节筛选、顺序/随机练习
- 支持单选、多选、判断、简答题
- 错题本与学习进度本地存储
- 学生门禁登录（手机号+邀请码，学号可选）
- 登录设备上报（用于后端设备上限控制）

使用方法：

1. 打开微信开发者工具。
2. 导入项目目录 `wechat-miniprogram`。
3. 在 `project.config.json` 替换 `appid` 为你自己的 AppID。
4. 在 `config.js` 中设置后端地址 `API_BASE_URL`。
5. 预览运行并提交审核。
