# 老师后台使用说明

## 1. 启动后端

```bash
cd /Users/apple/Downloads/专升本/backend
cp .env.example .env
# 编辑 .env，至少设置：DATABASE_URL、JWT_SECRET、ADMIN_SECRET
npm install
npm run prisma:generate
npm run prisma:migrate -- --name add_admin_student_tools
npm run seed
npm run import:students -- --file=./scripts/students_template.csv
npm run start:dev
```

## 2. 打开老师后台页

- 打开：`/Users/apple/Downloads/专升本/teacher_admin.html`
- 推荐通过本地静态服务访问（避免浏览器 file:// 跨域限制），例如：

```bash
cd /Users/apple/Downloads/专升本
python3 -m http.server 5173
```

然后访问：`http://127.0.0.1:5173/teacher_admin.html`

## 3. 页面中填写

- `API Base`：`http://127.0.0.1:3000/api/v1`
- `Admin Secret`：你在 `.env` 设置的 `ADMIN_SECRET`

## 4. 可执行操作

- 加载/筛选学生
- 单条新增或更新学生
- 冻结/解冻账号
- 设置设备上限
- 清空某学生已绑定设备
- CSV批量导入白名单

## 5. CSV格式

表头必须包含：

`studentNo,name,phone,className,inviteCode,maxDevices,expiresAt,status`

示例：

`20260001,张三,13800000001,心理学1班,PSY2026A,1,2026-12-31,ACTIVE`
